OVERVIEW
--------
FleetLabs has 5 live sections + Command Center dashboard:
  1. Command Center    – KPI overview, radar, maps, heatmaps
  2. Inventory Scanner – Gemini 2.5 Flash + YOLO object detection
  3. Load Intelligence – REKA Edge + YOLO live load estimation
  4. Damage Control    – REKA Edge + YOLO damage triage
  5. Last-Mile Risk    – Route risk scoring with weather + LLM
  6. Agent Monitor     – Gemini 2.5 Flash HITL (Human-in-the-Loop) decisions

DEMO ORDER (follow sidebar top→bottom): 
  Command Center → Inventory → Load → Damage → Last-Mile → Agent Monitor

================================================================================
  TIPS FOR MAXIMUM WOW FACTOR
================================================================================

  1. ALWAYS seed demo data before starting (inventory page → "Seed Demo" button)
    2. For Inventory Scanner: use images with DIVERSE, BRANDED objects (not just bottles)
      - Gemini is the final extractor; YOLO is only the visible operator overlay
  3. For Load Intelligence: the LIVE CAMERA mode is your single biggest wow moment
     - Put a box on the table, add/remove items while camera watches
  4. For Damage Control: keep TWO images ready — damaged + pristine (shows no FP)
  5. For Agent Monitor: let events stream live, approve/reject in real-time
  6. Don't rush — let animations complete (gauge fills, events streaming)
  7. Keep browser at 90% zoom for wider dashboard view
  8. If backend/API is slow, say "processing through our AI pipeline" 
  9. Have a backup: if live demo fails, seed data makes all pages still look populated
